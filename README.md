
## 프로젝트 실행경로
`devd -> test -> my-app`

## 실행환경
- `@types/node` ^16.18.61
- `@types/react` ^18.2.37
- `react` ^18.2.0`
- `npm` 9.5.1
    
## 패키지 설치
`npm install`

## 프로젝트 시작
`npm start`
